# Welcome to ImageLabelling-UI

Browse example src now by navigating to them in the sidebar. View their code in the `src` directory to learn how they work. We recommend building UIs with a [**component-driven**](https://componentdriven.org) process starting with atomic components and ending with pages.

[**The atomic design methodology**](https://atomicdesign.bradfrost.com/chapter-2/#the-atomic-design-methodology) is comprised of 5 discrete levels: [atoms](https://atomicdesign.bradfrost.com/chapter-2/#atoms), [molecules](https://atomicdesign.bradfrost.com/chapter-2/#molecules), [organisms](https://atomicdesign.bradfrost.com/chapter-2/#organisms), [templates](https://atomicdesign.bradfrost.com/chapter-2/#templates), and [pages](https://atomicdesign.bradfrost.com/chapter-2/#pages).
